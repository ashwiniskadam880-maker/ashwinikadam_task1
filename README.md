Java Reservation System

Overview

This is a console-based Railway Reservation System developed in Java. The project allows users to log in, reserve train tickets, cancel reservations, and exit the system through a menu-driven interface.

Features

- User Login Authentication
- Train Ticket Reservation
- Automatic Train Name Selection Based on Train Number
- Passenger Details Management
- Reservation Cancellation
- Menu-Driven Interface
- Exit Option

Technologies Used

- Java
- Scanner Class
- Conditional Statements (if-else)
- Loops (while)
- String Handling

Working

1. User logs in with username and password.
2. After successful login, a menu is displayed.
3. Users can create a reservation by entering passenger and train details.
4. Users can cancel reservations using the cancellation option.
5. The system continues running until the user selects Exit.

Learning Outcomes

- Understanding of Java fundamentals
- User input handling
- Menu-driven programming
- Conditional logic and loops
- Basic railway reservation system implementation
